# NetPop

Requirements : 
 - Python 3-3.7.1
 - Flask-1.0.2
 - PyMySQL-0.9.3
 - Flask-WTF-0.14.2
 - passlib-1.7.1
 - flask-executor-0.8.3 


Ranks :
 - 1  Guest
 - 2  Monitor - Monitor Page Access Only (Default)
 - 3  Administrator (Settings Pages and must be set manually in the db)

 To Run : ``` python __init__.py ```
